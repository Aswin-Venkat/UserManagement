User Management Application

Libraries used:

1.React
2.Redux
3.Redux Form
4.Enzymes
5.React Router

To run app: npm install and npm start

Launch url:  http://localhost:3000/#/

To run unit test: npm test

To build app : npm build

Functionalities built:

1. Login
2. Registration
3. Users dashboard
4. Add user
5. Remove user
